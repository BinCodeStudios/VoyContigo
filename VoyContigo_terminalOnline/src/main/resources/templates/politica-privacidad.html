1. Recopilación de Información

Recopilamos datos personales como nombre, correo electrónico y número de identificación, cuando te registras, compras un tiquete o accedes a funcionalidades específicas de nuestra plataforma.

2. Uso de la Información

La información recopilada se utiliza para brindar un servicio seguro y personalizado, mejorar la experiencia del usuario, emitir facturas, y cumplir con obligaciones legales o contractuales.

3. Protección de Datos

Implementamos medidas de seguridad avanzadas para proteger tus datos contra accesos no autorizados, alteraciones o destrucción. Nuestra base de datos está cifrada y auditada periódicamente.

4. Compartición de Datos

No vendemos ni compartimos tu información con terceros, salvo cuando sea estrictamente necesario para cumplir con requerimientos legales o prestar servicios relacionados con el transporte.

5. Derechos del Usuario

Como usuario tienes derecho a acceder, modificar o eliminar tus datos personales, así como a revocar tu consentimiento. Para ello, puedes escribirnos a datos@voycontigo.com.

6. Cambios a esta Política

Nos reservamos el derecho de modificar esta política en cualquier momento. Las actualizaciones serán notificadas en esta misma página con su respectiva fecha de modificación.
